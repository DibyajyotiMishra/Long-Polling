# Multi-user Long Polling App

This is a simple multi-user long polling app.

## Tech Stack
- Node.js
- Express.js
- Redis (switch to redis branch to see the implementation)
- HTML
- CSS
- JavaScript

## Features:
- Long Polling
- Delivery Guarantee
- Multi-user Support
- Admin Dashboard to view live stats
- Data Persistence using Redis (switch to redis branch to see the implementation)
